# Graphics - Semester IV - UBB

The repository contains the seminar and laboratory exercises from the Graphics course.

The main focus is OpenGL programming and creating 2D/3D models.

**Projects included:**

- Rubik's Cube: The user can shuffle the cube with 30 random moves and attempt to solve it manually. The application supports 3D camera movement and the cube sides can be rotated using the keyboard or UI buttons.
- Foosball Simulation: A game where the user drives a ball between moving players on a football field. The features includes dynamic player movement and the ability to toggle between three different camera views.
